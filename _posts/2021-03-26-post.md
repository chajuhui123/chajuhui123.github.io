# 마크다운?

- 마크다운이란...

    마크다운(markdown)은 일반 텍스트 문서의 양식을 편집하는 문법이다. README 파일이나 온라인 문서, 혹은 일반 텍스트 편집기로 문서 양식을 편집할 때 쓰인다. 마크다운을 이용해 작성된 문서는 쉽게 HTML 등 다른 문서형태로 변환이 가능하다.     *위키백과*

<br>

추후 <깃허브>를 사용하면서 자주보게 될 README.md!
개발자들이 자신의 프로젝트에 대해 정리해놓은 텍스트 문서라고 이해하면 EZ

<br>

## VScode 용 마크다운 확장기능 설치

1. 좌측 맨 아래 <마켓플레이스> 아이콘 클릭
2. <markdown all in one> 검색
3. install
4. 작업폴더 열고
5. <redme.md> 파일 생성
6. 우측 상단 <측면에서 미리보기> 아이콘 클릭
7. 준비완료

<br>

## 마크다운 문법 살펴보기

### 헤딩

```css

# 헤딩1
## 헤딩2
### 헤딩3
#### 헤딩4
##### 헤딩5
###### 헤딩6
```

### 결과

# h1

## h2

## h3

<br>

### 강조, 이탤릭체

```css
*이탤릭체*
**굵게**
**_강조 및 이탤릭체 혼합_**
```

### 결과

*이탤릭체*

**굵게**

***강조이탤릭체혼합***

<br>

### 수평선

```css
---
___
***
```

### 결과

***

***

<br>

### 코드블럭

```css
```
코드작성
```
```

### 결과

```css
코드작성
```

<br>

### 이미지업로드

```css
![이미지설명](이미지주소)

![멋사]([https://startup.likelion.org/img/logo.png](https://startup.likelion.org/img/logo.png))
```

### 결과

![멋사]([https://startup.likelion.org/img/logo.png](https://startup.likelion.org/img/logo.png))

<br>

### 리스트

- 번호가 있는 리스트

```css
1. 내용
2. 내용
```

- 번호가 없는 리스트

```css
* 1뎁스
	* 2뎁스 (tap)
```

### 결과

1. 내용
2. 내용

- 1뎁스
    - 2뎁스

<br>

### 인용구

```css
> 인용구
```

### 결과
