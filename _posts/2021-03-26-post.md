# 마크다운?

- 마크다운이란...

    마크다운(markdown)은 일반 텍스트 문서의 양식을 편집하는 문법이다. README 파일이나 온라인 문서, 혹은 일반 텍스트 편집기로 문서 양식을 편집할 때 쓰인다. 마크다운을 이용해 작성된 문서는 쉽게 HTML 등 다른 문서형태로 변환이 가능하다.     *위키백과*

     

추후 <깃허브>를 사용하면서 자주보게 될 README.md!

개발자들이 자신의 프로젝트에 대해 정리해놓은 텍스트 문서라고 이해하면 EZ

### 헤딩

```css

# 헤딩1
## 헤딩2
### 헤딩3
#### 헤딩4
##### 헤딩5
###### 헤딩6
```

<결과>

# h1

## h2

## h3

...

### 강조, 이탤릭체

### 리스트

번호가 있는 리스트

```css
1. 내용
2. 내용
```

번호가 없는 리스트

```css
* 1뎁스
	* 2뎁스 (tap)
```

<결과>

1. 내용
2. 내용

- 1뎁스
    - 2뎁스
